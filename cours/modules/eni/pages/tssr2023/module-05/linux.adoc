== Partitionnement

découpage du disque dur `sda` en 2 partie:

* sda1 Partition lvm :  18 176 MB
* sda2 partition lvm :  le reste

[cols="^.^,^.^,^.^"]
|===
3.+| Volume Groupe (lvm)
| label | Partition  |Site (MB)
| system | sda1  | *18 176MB*
| data | sda2  | Le reste
|===
